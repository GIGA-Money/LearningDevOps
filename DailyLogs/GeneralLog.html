  • Large instruction set Model
  • What if we had and large language model that could be used to generate agent unique instructions.
  • Consider the implications on Stock automation agents, game and simulation agents.
  • We could just have a tool that would take request, instructions, actions, and other written formats
    then deploy an agent to act out with language rather than produce a arithmetic or algorithmic solution.

    Such as in a code format.
  • An ideal way for implementation would be an Computer Only interface language, COIL.
